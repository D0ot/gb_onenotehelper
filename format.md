Data Format

UShort      :       Infomation, touch, hovering, etc.
UShort      :       x pos
UShort      :       y pos
UShort      :       height, distance between pen to screen
UShort      :       pressure
Int8        :       x rorate, left for negative
Int8        :       Unknown
Int8        :       y rorate, up for negative


example :

0x0221 3F3E B736 7D09 0A54 0B540B
